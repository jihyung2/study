스택 : 후입선출 구조, 삽입과 삭제가 한 쪽에서만 일어나는 특정

스택 용어 :
    top : 삽입과 삭제가 일어나는 위치를 뜻한다.
    push : top 위치에 새로운 데이터를 삽입하는 연산
    pop : top 위치에 현재 있는 데이터를 삭제하고 확인하는 연산
    peek : top 위치에 현재 있는 데이터를 단순 확인하는 연산

스택은 우선 탐색, 백트래킹 종류에서 자주 사용, 재귀 함수 알고리즘 원리와 비슷

큐 : 선입선출 구조, 삽입과 삭제가 양방향에서 일어나는 특징

큐 용어 :
    rear : 큐에서 가장 끝 데이터를 가리키는 영역
    front : 큐에서 가장 앞의 데이터를 가리키는 영역
    add : rear부분에 새로운 데이터를 삽입하는 연산
    poll : front부분에 있는 데이터를 삭제하고 확인하는 연산
    peek : 큐의 맨 앞에 있는 데이터를 확인할 때 사용하는 연산

                      peek
     rear             front
    ->add| | | | | | |->poll

큐는 너비 우선 탐색에 자주 사용함

우선순위 큐 : 값이 들어간 순서와 상관없이 우선순위가 높은 데이터가 먼저 나오는 자료구조
