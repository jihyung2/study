구간 합은 합 배열을 이용하여 시간 복잡도를 더 줄이기 위해 사용하는 특수한 목적의 알고리즘

코딩테스트에서의 빈도가 매우 높음

합 배열 S의 정의
S[4] = A[0] + A[1] ... + A[3]

합 배열 S를 만드는 공식
S[i] = S[i-1] + A[i]

부분 구간 합을 구하는 공식
i에서 j까지 구간합을 구하는 공식
S[j] - S[i-1]

받는 데이터가 많을 때는 Scanner보다 BufferedReader사용하기, 더 빠르기 때문임


투 포인터는 2개의 포인터로 알고리즘의 시간 복잡도를 최적화 하는 방법이다.
알고리즘이 매우 간단하므로 바로 실전문제로 익혔다.
